# ActantDB Naming and Vocabulary Rules

- **actantdb**: Lowercase, single word when referring to the product.
- **actantdb (CLI)**: Name of the CLI binary.
- **@actantdb/all**: Name of the umbrella npm package.
- **actant-* (crates)**: Crates must be named in kebab-case.
- **Actant* (Rust types)**: Rust types must be written in PascalCase.
- **snake_case**: SQL identifiers must use snake_case.
- **TEXT**: SQL IDs must be TEXT fields.
- **ISO-8601**: Timestamps must be ISO-8601 strings.
