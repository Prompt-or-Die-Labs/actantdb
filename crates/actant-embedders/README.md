# actant-embedders

Embedder + reranker provider registry. FastEmbed (default local, ONNX), MLX (Apple silicon), llama.cpp embeddings, OpenAI, Voyage AI (with Voyage 4 shared embedding spaces), Cohere, Jina, Mixedbread, Nomic, any OpenAI-compatible endpoint. Sparse encoders (SPLADE, BM42). Local + cloud cross-encoder rerankers. Versioned model metadata + dimension + distance-metric tracking.

See `agents/actant-embedders.md`.
