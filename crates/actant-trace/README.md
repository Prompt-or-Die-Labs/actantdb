# actant-trace

OpenTelemetry GenAI + OpenInference compatible emitters across every ActantDB subsystem. Span kinds: model_call, tool_call, retrieval, reranker, embedding, agent, workflow, approval, replay. Metrics for queue depth, model latency, token usage, retry counts, budget remaining, cache hit rate. Exports to OTLP collectors; Phoenix/Arize/LangSmith/Datadog/Grafana compatible.

See `agents/actant-trace.md`.
