# actant-protocol

Protocol adapters: MCP (Model Context Protocol — resources, prompts, tools), A2A (Agent-to-Agent — agent cards, capability negotiation, secure delegation), AP2 (Agent Payments Protocol — verifiable intent, mandates, cryptographic audit). Each protocol's records become first-class rows so cross-agent and commerce interactions enter the causal graph.

See `agents/actant-protocol.md`.
