## Pattern legend

- **process-spawn** — iOS sandbox forbids spawning arbitrary binaries (posix_spawn to non-bundled bins).
- **system-libsqlite** — System libsqlite linkage; iOS must use sqlx with the bundled feature (default).
- **home-dir** — Hard-coded $HOME path; iOS caller supplies the sandbox dir explicitly.
- **tilde-actantdb** — Hard-coded ~/.actantdb path; iOS caller supplies the sandbox dir explicitly.
- **native-tls** — Should use rustls-tls on iOS to avoid Security.framework weight.
- **fork-exec** — iOS sandbox forbids fork/exec.
- **process-ids** — Process-table reads are sandboxed on iOS; wrap in #[cfg(not(target_os = "ios"))].
- **tempfile-no-dir** — tempfile without an explicit dir argument falls back to /tmp; iOS needs the app sandbox tempdir.
